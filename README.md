> A Vue.js 2.0 UI Toolkit for djzy Web
## Install
```shell
npm install dajia-ui -S
```
## Quick Start
``` javascript
import Vue from 'vue'
import DajiaUi from 'dajia-ui'
Vue.use(DajiaUi)
import {
  Toast,
  Alert
  // ...
} from 'DajiaUi'
Vue.component(Toast.name, Toast)
Vue.component(Alert.name, Alert)
```
## Browser Support
Modern browsers and Internet Explorer 10+.
